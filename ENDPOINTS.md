# API Endpoints - Monitoreo de Inundaciones

Base URL: `http://localhost:8000/api`

---

## Municipios
- GET `http://localhost:8000/api/municipios` → Listar todos los municipios
- GET `http://localhost:8000/api/municipios/{id}` → Obtener un municipio por id
- POST `http://localhost:8000/api/municipios` → Crear un nuevo municipio
- PUT `http://localhost:8000/api/municipios/{id}` → Actualizar un municipio
- DELETE `http://localhost:8000/api/municipios/{id}` → Eliminar un municipio
